If your app has the need for a custom authentication flow, the Carol FE SDK provides methods to help with the password recovery flow.


The password recovery flow starts with a recovery request, this request generates a token that is sent to the target e-mail address.
With the token in hand, the user can is redirected to a recovery page, where a new password is chosen and a reset request is sent with the authentication token

Use the form below to send a password recovery request using the CarolAuthService methods.
Follow the link sent to your e-mail address to be redirected to the Reset Password tab with a valid token.
